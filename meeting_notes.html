Client’s Feedback on Requirement Specifications

“For project scope, the main priority should be the development of a mobile application”
“All user types (customers and workers) must be able to browse menus”
“The ability to leave notes in regards to food allergies is a must”
“All users regardless of payment methods log in: via their UVic account or create a guest account”
“Supporting customer’s friends picking up their order should be disregarded for the risks it presents”
“No mapping function is needed as these already exist through UVic’s website”
“Workers must have a way to update the menus, to add new items or mark them as sold out”
“We can limit the language to English only”
“As the system is a mobile app, deploying to both Android and iOS is essential”
